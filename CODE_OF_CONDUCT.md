# Code of Conduct

## The Archmage's Oath

As practitioners of the Motus Magic System, we commit to:

1. **Welcome all skill levels.** Every archmage was once an apprentice.
2. **Assume good intent.** Misunderstandings are not attacks.
3. **Teach generously.** If someone doesn't know, show them.
4. **Disagree respectfully.** Challenge ideas, not people.
5. **Give credit.** Name your sources. Acknowledge contributions.
6. **Keep it safe.** No harassment, discrimination, or intimidation.

## Enforcement

Violations can be reported to InitiumBuilders@gmail.com. We will respond within 48 hours.

## The Simple Version

Be the kind of mage you'd want to learn from.
