The 80/20 of Cyber Defense

Five things that stop the majority of attacks. No exceptions. Do these first.

⚡ CRITICAL
Enable MFA Everywhere

Blocks 99.9% of account compromise attacks. Email, banking, social, cloud — all of it. Use an authenticator app (not SMS). This single action has more security ROI than anything else on this page.

⚡ CRITICAL
Patch Within 48 Hours

Most exploits target known CVEs — vulnerabilities that already have patches. Attackers weaponize new CVEs within 24-48 hours of disclosure. Patch fast. Automate it where you can. The attacker is waiting for you to delay.

⚡ HIGH
Principle of Least Privilege

Every user, every service, every process gets exactly the permissions it needs — nothing more. When an attacker compromises an account, they inherit its permissions. Minimize the blast radius before the breach happens.

⚡ HIGH
Network Segmentation

Divide your network into zones. If an attacker gets into one zone, they shouldn't be able to reach everything else. Stop lateral movement before it becomes a full breach. Critical for both enterprise and home networks.

⚡ HIGH
The 3-2-1 Backup Rule

3 copies of your data. 2 different media types. 1 offsite. This is ransomware's kryptonite. If you have a clean, verified backup from yesterday, ransomware is just an inconvenience — not a catastrophe.

The Defender's Hacks

Techniques the pros use. Cheap to implement. Highly effective. Often overlooked.

✅ FREE
Canary Tokens — Invisible Tripwires

Free at canarytokens.org. Drop fake files, URLs, or credentials in sensitive locations. The moment an attacker opens them, you get an alert. Set it and forget it. The best intrusion detection you've never deployed.

⚡ HIGH
DNS Sinkholing

Redirect known malicious domains to localhost (127.0.0.1). At the network level, this kills C2 callbacks, malware downloads, and phishing pages before they load. Use a regularly updated blocklist. Pihole + block lists = instant upgrade.

✅ FREE
Fail2ban + Rate Limiting

Fail2ban watches your logs and bans IPs after repeated failures. SSH brute force, login attempts, web scrapers — all get automatically blocked. Stops 99% of automated attacks at the door. Install it in 5 minutes.

⚡ CRITICAL
SSH Keys Only — No Passwords

Disable SSH password authentication entirely. Key-based auth is cryptographically stronger, immune to password spraying and brute force, and requires no password to forget or reuse. One config line eliminates an entire attack category.

⚡ HIGH
Immutable Infrastructure

Don't patch compromised servers — rebuild them from a clean image. When something is wrong, nuke it and restore from code. Attackers love persistent access; immutable infra denies it. Infrastructure-as-code + short-lived VMs = this for free.

The Mental Models

How to think about security. These frameworks change your instincts.

Assume Breach

Design every system assuming the attacker is already inside. What can they see? What can they move to? What can they exfiltrate? Build layers so that being inside one zone doesn't mean being inside all zones. Paranoia as architecture.

Defense in Depth

No single security control is perfect. Layer them. If the firewall fails, the EDR catches it. If the EDR fails, the backup saves you. If the backup fails, the insurance pays out. No single point of failure. The attacker must win every layer.

Zero Trust

Never trust, always verify. Don't trust anything by default — not the device, not the network, not the user. Verify every access request every time. "Inside the perimeter" is not a meaningful concept anymore. Act accordingly.

Shift Left

Security bugs found in design cost 1x. Found in development: 10x. Found in production: 100x. Shift security left into the design phase, the sprint planning, the code review. The earlier you catch it, the cheaper it is to fix.

Blast Radius Thinking

For every system you build, ask: if this is compromised, how far can the blast reach? What data is exposed? What other systems are reachable? Minimizing blast radius is the meta-skill behind least privilege, segmentation, and isolation.

Free Power Moves

Professional-grade tools. All free. Use them now.

✅ FREE
Have I Been Pwned

Check if your email and passwords have been leaked in data breaches. haveibeenpwned.com — run by Troy Hunt, trusted by millions. Set up email alerts for new breaches. Do this today.

✅ FREE
VirusTotal

Scan any file or URL against 70+ antivirus engines instantly. virustotal.com — before you open that attachment, before you visit that link. 5-second habit that catches real threats.

✅ FREE
Shodan — See What's Exposed

shodan.io — search engine for internet-connected devices. Type your IP or domain and see what the world can see. Cameras, routers, databases, APIs — all indexed. If an attacker can find it, so can you.

✅ FREE
SecurityHeaders.com

Test your website's HTTP security headers instantly. securityheaders.com — grades you A through F, tells you exactly what's missing. CSP, HSTS, X-Frame-Options — add these and stop an entire class of web attacks.

✅ FREE
SSL Labs TLS Test

Deep analysis of your TLS configuration. ssllabs.com/ssltest — checks your certificate chain, cipher suites, protocol versions, and known vulnerabilities. Get an A+. It's free and takes 30 seconds.

✅ FREE
Mozilla Observatory

Comprehensive web security scan. observatory.mozilla.org — tests headers, TLS, cookies, redirects, and more in one shot. Mozilla-maintained, trusted, free. Run it on any site you own.

The Insider Knowledge

What practitioners know that most people don't. The uncomfortable truths.

Phishing Starts 80%+ of Breaches

Most security budgets go to firewalls and EDR. Most breaches start with a phishing email. Security awareness training — real, ongoing, phishing-simulation training — has more ROI than almost any technical control. Train the humans.

Cloud Misconfig = #1 Cloud Attack Vector

Public S3 buckets. Open database ports. Overpermissioned IAM roles. Not clever exploits — configuration errors made under pressure. Audit your cloud permissions regularly. Use CSPM tools. Assume your team made mistakes because teams always do.

Supply Chain Attacks Are Growing Fast

SolarWinds. Log4j. XZ Utils. Attackers compromise one trusted library and reach millions of systems. Verify your dependencies. Use SBOM. Pin versions. Review what you're importing. The thing you didn't write is often the thing that gets you.

Ransomware Groups Operate Like Startups

Ransomware-as-a-Service (RaaS). Bug bounties for vulnerabilities. Customer service portals. Negotiation teams. This is a mature industry with professional operations. Your adversary is organized, funded, and motivated. Plan accordingly.

AI Phishing Is Indistinguishable From Human

AI-generated spear-phishing emails now perfectly mimic writing style, use personal context from LinkedIn, and craft contextually accurate lures. You cannot trust the quality of writing as a signal. Verify high-stakes requests via a separate channel. Always.
